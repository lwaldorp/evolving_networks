# evolving_networks
R-code for the paper "Evolving networks, Markov chains and dynamical systems: Unifying principles" by Lourens Waldorp and Maarten Marsman

There are four separate files for each of the different parts in the paper

- network.R # for the network simulations
- markov-chain.R # for the Markov chain simulations
- dynamical-system.R # for the dynamical system plots
- metropolis-markov-ising.R # for the data analysis

Each file is annotated and contains library statements for dependencies.
